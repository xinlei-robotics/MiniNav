# V0: Ideal Motion Simulation

## Goal

Implement a minimal 2D robot simulator:
- state: x, y, yaw
- control: v, w
- discrete-time propagation
- trajectory export

## Why this matters

This is the foundation for:
- odometry
- state estimation
- EKF prediction model
- later path tracking

## Output

- `data/traj.csv`
- trajectory plot in `results/`
